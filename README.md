# Required pkgs 
(known)

- I3  
	Window manager
- Conky  
	Text on wallpaper
- polybar  
	Main bar
- Emptty  
	Session Manager
- Vim  
	Text editing (<3)

To install:
```
sudo xbps-install -Syu i3 conky polybar emptty vim  
```
